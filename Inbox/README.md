# Inbox

This folder is a temporary holding area for information that hasn't been classified yet.

## Purpose
When the AI Classifier agent isn't confident about where something belongs (confidence < 70%), it will place items here for you to manually review and organize later.

## Workflow
1. Review items in Inbox periodically
2. Move them to the appropriate PARA folder (People, Projects, Areas, Resources)
3. Or archive/delete if not needed
4. Keep Inbox empty or near-empty

## Tips
- Process inbox weekly to avoid buildup
- If you find recurring patterns, provide examples to improve the Classifier agent
- Items older than 30 days may no longer be relevant
